# Vel'thara Reference — Table of Contents

> **Source:** Homebrew — Duskport Campaign, Session 8+
> **Usage:** DM reference for Vel'thara embassy interactions, cultural exchange arc, and cross-threshold diplomacy.
> **Key principle:** The Vel'thara are not monsters, not gods, not humans-in-costume. They are genuinely alien beings who experience reality as frequency. Play them as curious, patient, kind — and fundamentally different.

2. One Speaker at a time
3. Shared garden (moonpetals, neutral ground)
4. Exchange, not extraction
5. Kill switch with the Warden (Zaryth)
6. Names are sacred (Dawnwatch ↔ Vel'thara)
