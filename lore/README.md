# Lore — Campaign World Reference

## Contents

| File | Description |
|------|-------------|
| `calendar-guide.md` | 12-month calendar (360-day year, 7-day weeks). Seasons, weather, festivals, Dawnwatch traditions, moon phases (Selara). |
| `history-known-world.md` | 3,000-year world history — Age of Binding, Covenant Kingdoms, Fracture Wars, the Sundering, Reconstruction, Modern Age. Told by Vaelith + Seren. |
| `library-catalogue.md` | Dawnwatch library inventory — 20 books (14 acquired + 6 authored). Planned volumes. |

| `trees-management.md` | Orchard & shade tree management — 35 trees, growth acceleration (druid + divine + ward), planting plans, harvest projections, soil amendments. |
| `weather-climate.md` | Regional weather & climate by location and season. Sea state, sailing windows, weather hazards, Underdark climate, crop growing conditions. |
| `drow/` | Drow reference guide (5 files) — society, military, magic, stat blocks, Underdark geography. |
