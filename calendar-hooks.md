## How to Use This File

1. **Start of session:** Read current day from `party-state.json`. Check this file for any events between last session's day and now.
2. **Before time-skip:** Scan all events in the skip range. Flag any that should be played out.
3. **After session:** Update one-time events (mark complete or adjust dates). Add new hooks discovered during play.
4. **DM prompt:** "Check calendar-hooks.md for events between Day X and Day Y before advancing."
